MindMapr – Smart Revision Assistant

This is a Python console application for study topics revision.

Menu:

  1. Add Topic
  2. View All Topics
  3. Search Topic
  4. Quiz Mode
  5. Exit

Example Topics (ECE):

  • Digital Electronics: Logic Gates, Flip-Flops, Multiplexers, Decoders
  • Microprocessors: 8085 Architecture, Instruction Set, Interrupts
  • Communication Systems: AM, FM, Modulation, Demodulation, Noise
  • Signal Processing: Fourier Transform, Laplace Transform, Filters
  • Control Systems: Open-loop, Closed-loop, Stability, PID Controller
  • VLSI: CMOS, Fabrication, Scaling, Design Rules
  • Embedded Systems: Microcontrollers, Real-Time OS, Sensors

Note: To use the app, run mindmapr.py in a Python environment.
